
# 💰 Personal Finance Tracker

## 📌 Project Description
The Personal Finance Tracker is a simple application designed to help users track their monthly income, expenses, and investments in an organized way. It provides a basic overview of personal financial activity and helps improve money management habits.

---

## 🚀 Features
- Add and track monthly income
- Record monthly expenses
- Track investments separately
- View overall financial summary
- Simple and beginner-friendly structure

---

## 🛠️ Technologies Used
- HTML
- CSS
-  JavaScript
- Basic data handling and logic
- File storage or local data management
